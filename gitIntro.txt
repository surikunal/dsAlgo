to add user name and email of github account :
 $>	git config --global user.name "Yash962000"
 $>	git config --global user.email "yashsaxena962000@gmail.com"

to configure editor for writing commit message :
 $>	git config --global core.editor "notepad.exe"
//  default vim

to make a folder as a repository :
 $>	git init .

to check status of commits and branch master :
 $>	git status

to add file in repository :
 $>	git add .
or $>	git add <filename>

to check log of commits :
 $>	git log
or $>	git log --oneline
//  this one is more compressed

to commit all the changes :
 $>	git commit
//  to exit vim editor , ctrl + s	then ":wq"
//  if commit message is of 1 line
 $>	git commit -m "message for git commit"
